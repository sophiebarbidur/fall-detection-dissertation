# Exploratory Analyses

These scripts were used during development to diagnose per-subject failure modes. They are not directly referenced in the dissertation but informed the failure-mode taxonomy in Chapter 5.3.

## Files

- `signal_diagnostics.py` — computes per-subject signal statistics, z-score deviation from the cross-subject mean, and Jensen-Shannon divergence. Identifies S10 as a signal outlier.
- `lying_vs_falls_analysis.py` — compares motion energy and waveform shapes across correctly classified falls, correctly rejected lying windows, and misclassified lying windows.
- `failing_subject_waveforms.py` — compares fall waveforms of failing subjects (S07, S10, S13, S16) to the training pool for specific fall types, diagnosing the under-energy and over-energy failure modes.

## Prerequisites

These scripts require `loso_per_window_predictions.csv` in `outputs/diagnostics/`, which is generated by `notebooks/failure_mode_diagnostics.py`.
